
# ZaykaNow

**ZaykaNow** is a modern food ordering frontend application built with **React** and **Tailwind CSS**. It allows users to browse food items by category, search for specific dishes, manage a cart, and simulate a successful orderâ€”all with a clean and responsive user interface.

## Features

- Browse food items by category
- Search for food items by name or keyword
- Add and remove items from the cart
- View cart details with real-time total calculation
- Simulate order placement
- Smooth, responsive UI with Tailwind CSS

## Tech Stack

- **Frontend Framework:** React
- **Styling:** Tailwind CSS
- **State Management:** React useState/useContext (or similar)
- **Cart Persistence:** Local state (or LocalStorage if implemented)
